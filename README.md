# Ironfan-Pantry

This is Infochimps' battle-hardened collection of cookbooks. Some are by us, some are forks, some are dupes. See the CREDITS below.

Documentation is at the [ironfan repo](https://github.com/infochimps-labs/ironfan)
To get started, clone the [ironfan-homebase repo](https://github.com/infochimps-labs/ironfan-homebase) and follow its instructions.
